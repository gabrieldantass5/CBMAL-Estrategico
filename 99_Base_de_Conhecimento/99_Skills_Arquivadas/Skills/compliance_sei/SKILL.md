# Compliance SEI (Sistema Eletrônico de Informações)

## Checklist de Pré-Inserção
1.  **Formato**: O documento está em PDF pesquisável (OCR) ou editor nativo do SEI?
2.  **Tamanho**: O arquivo é menor que 10MB? (Se maior, precisa de link externo ou compressão).
3.  **Legibilidade**: A digitalização está limpa e sem rasuras?
4.  **Assinatura**: O documento prevê espaço para assinatura digital? Não inclua "riscos" para assinatura manual.

## Classificação
*   **Público**: Regra geral.
*   **Restrito**: Documentos preparatórios, dados pessoais.
*   **Sigiloso**: Segurança de Estado (raro, exige justificativa).

## Nomenclatura no SEI
*   Use nomes descritivos na árvore de documentos.
*   Ex: `Despacho de Deferimento`, `Anexo - Relatorio Fotografico`.
*   Evite: `Doc1`, `Digitalização`, `Sem título`.
