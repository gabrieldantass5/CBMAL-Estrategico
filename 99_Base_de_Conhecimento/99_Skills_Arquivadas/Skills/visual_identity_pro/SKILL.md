# Visual Identity Pro

## Cores Oficiais CBMAL
*   **Vermelho CBMAL**: `#CC0000` (Aproximado para digital)
*   **Amarelo Ouro**: `#FFCC00`
*   **Azul Real** (Fardamento): `#003399`
*   **Branco**: `#FFFFFF`
*   **Preto**: `#000000`

## Tipografia
*   **Títulos**: Montserrat ou Roboto (Bold).
*   **Corpo**: Inter, Open Sans ou Arial.

## Elementos Gráficos
*   **Brasão**: Sempre no canto superior esquerdo ou centralizado no topo.
*   **Rodapé**: Endereço da unidade e contato.
*   **Marcadores**: Use quadrados ou círculos sólidos. Evite setinhas informais.

## Hierarquia Visual
*   **Título 1**: Tamanho 24, Negrito, Vermelho ou Azul.
*   **Título 2**: Tamanho 18, Negrito, Preto.
*   **Título 3**: Tamanho 14, Itálico, Cinza Escuro.
*   **Corpo**: Tamanho 12, Preto.
